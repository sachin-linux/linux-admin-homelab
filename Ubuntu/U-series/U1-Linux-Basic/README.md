#U1 - Linux Basic

#Objective 
Understand fundamental Linux commands and system navigation for system administration.

## Topics Covered

- Linux filesystem structure
- Basic shell commands
- File and directory management
- Viewing and editing files
- File permissions basics
- Process monitoring basics

## Key Commands

pws
ls
cd
cp 
mv
rm
mkdir
touch
cat
less
nano
head
tail 

## Filesystem Navigation

Examples: 

pwd
ls -l
cd /var/log
cd ~

## File Operations

Create file 

touch file.txt

copy file

cp file.txt backup.txt

Move file

mv file.txt /tmp

Delete file

rm file.txt

## Lab

1. Create a directory named 'linux-practice'
2. Create files inside it
3. Copy and move files
4. Remove test files

## Skills Gained

- Basic Linux command usage
- Filesystem navigation
- File management operation
